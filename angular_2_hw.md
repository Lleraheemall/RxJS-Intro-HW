# Завдання: Керування списком курсів через BehaviorSubject

## 📝 Опис

Потрібно реалізувати сторінку керування курсами, де користувач може переглядати список, додавати нові курси та видаляти існуючі. Для зберігання та оновлення списку потрібно використати `BehaviorSubject`.

---

## 🎯 Ціль

- Навчитись працювати з `BehaviorSubject`
- Реалізувати реактивне оновлення списку курсів
- Додати можливість створення та видалення елементів
- Показувати актуальний список без перезавантаження сторінки

---

## ⚙️ Що потрібно зробити

1. Створити сторінку `CoursesManagerPage`
2. Вивести список курсів
3. Створити сервіс для роботи з курсами
4. Зберігати список курсів у `BehaviorSubject`
5. Надати можливість отримувати список як `Observable`
6. Додати форму створення нового курсу:
   - `title`
   - `category`
   - `duration`
7. Реалізувати додавання нового курсу в список
8. Реалізувати видалення курсу зі списку
9. Після додавання або видалення список повинен оновлюватись автоматично

---

## 📌 Умови

- Курс має містити:
  - `id`
  - `title`
  - `category`
  - `duration`
- Мінімум 3 стартові записи в тестових даних
- `id` для нового курсу має бути унікальним
- Список курсів має оновлюватись через `BehaviorSubject`
- Для відображення списку використати `Observable`

---

## 📤 Результат

Після виконання:

- Сторінка відображає список курсів
- Новий курс можна додати через форму
- Курс можна видалити зі списку
- Після змін список оновлюється автоматично
- Для реалізації використано `BehaviorSubject`
- Робота виконана в тому ж репозиторії, що і попереднє завдання
